# Retail Inventory SQL Analytics for PopBeauty

## Recommended application use
Use this as the #1 headline project for TikTok Shop Strategy & Analytics, TikTok Shop Business Analytics and TikTok Shop EMEA Data Science.

## Summary
Designed a relational SQLite database and used SQL/Python analysis to diagnose retail inventory imbalance across stores, products, warehouses, sales and shipments. The project identified understocked high-demand stores, overstocked boutiques, premium product misallocation, expiry risk, reorder threshold problems, shipment bottlenecks, discounting issues and gross profit differences by store tier.

## Skills shown
SQL, relational database design, ER modelling, retail analytics, inventory analysis, gross profit analysis, stockout diagnosis, operational recommendations, business storytelling.

## Files included
- `popbeauty_sql_business_analytics_report.pdf` — final business report.
- `popbeauty_sql_database_analysis.ipynb` — SQL/Python analysis notebook.
- `popbeauty_er_diagram.pdf` — database relationship diagram.
- `popbeauty_retail_database.db` — SQLite database.
