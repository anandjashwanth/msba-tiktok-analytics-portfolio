# PopBeauty SQL Database and Retail Inventory Analysis

- Notebook: `popbeauty_sql_database_analysis.ipynb`

## Overview

SQL-based business analytics project for a fictional UK beauty retailer. The notebook connects to a SQLite database, explores stores, products, categories, sales, shipments, and inventory, and diagnoses inventory imbalance across the retail network.

## What the notebook contains

This notebook contains **88 cells**: 60 markdown cells, 28 code cells, and 35 saved outputs. The HTML view was generated from the existing notebook content so reviewers can inspect the work without relying on GitHub's notebook preview.

## Methods and tools shown

- SQLite database connection and SQL querying
- Pandas-based query outputs
- Retail sales and stock analysis
- Store size and regional comparison
- Inventory imbalance and reorder analysis
- Business recommendation development
