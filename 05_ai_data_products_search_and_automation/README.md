# AI Data Products, Search and Automation

## Recommended application use
Use these as supporting technical projects. They are especially useful if the application allows more than three projects or asks for Python/AI examples.

## Summary
This folder contains applied AI projects covering RAG, semantic search, OCR, computer vision and business-facing AI workflows. For TikTok analytics roles, position these as evidence that you can turn analysis into repeatable tools and demos.

## Skills shown
RAG, semantic search, embeddings, FAISS, BM25, Gradio, Gemini Vision, OCR, prompt engineering, CNNs, computer vision, confidence scoring, business decision rules.

## Files included
- `ai_investment_rag_agent.ipynb` — AI investment evaluation RAG pipeline.
- `financial_news_semantic_search.ipynb` — financial news semantic search.
- `gemini_receipt_ocr_evaluation.ipynb` — Gemini Vision OCR extraction.
- `property_risk_scorecard_computer_vision.ipynb` — visual risk scorecard.
