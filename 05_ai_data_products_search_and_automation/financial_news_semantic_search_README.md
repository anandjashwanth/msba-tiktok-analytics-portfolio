## Overview

Semantic search project for financial news. The notebook cleans article text, extracts URLs, generates sentence embeddings, computes cosine similarity, retrieves relevant results for natural-language queries, and builds a Gradio interface.

## What the notebook contains

This notebook contains **13 cells**: 7 markdown cells, 6 code cells, and 25 saved outputs. The HTML view was generated from the existing notebook content so reviewers can inspect the work without relying on GitHub's notebook preview.

## Methods and tools shown

- Sentence-transformer embeddings
- Cosine similarity search
- Text cleaning and URL extraction
- Semantic retrieval for natural-language queries
- Gradio demo interface
- Financial-news information retrieval
