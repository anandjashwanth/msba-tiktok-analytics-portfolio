## Overview

Introductory binary classification notebook using a PyTorch multilayer perceptron on the German Credit dataset. The notebook loads public credit data, remaps the target, one-hot encodes categorical variables, scales features, creates DataLoaders, trains the model, and evaluates predictions.

## What the notebook contains

This notebook contains **13 cells**: 7 markdown cells, 6 code cells, and 8 saved outputs. The HTML view was generated from the existing notebook content so reviewers can inspect the work without relying on GitHub's notebook preview.

## Methods and tools shown

- PyTorch binary classification
- German Credit dataset
- One-hot encoding and scaling
- DataLoader workflow
- BCELoss and Adam optimiser
- Prediction versus actual comparison
