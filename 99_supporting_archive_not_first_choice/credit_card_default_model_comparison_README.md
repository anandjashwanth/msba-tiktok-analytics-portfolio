## Overview

Structured-data classification notebook comparing Random Forest, XGBoost, and TabPFN for credit-card default prediction. The notebook loads tabular data, separates features and target, preprocesses categorical variables, trains models, and evaluates classification performance.

## What the notebook contains

This notebook contains **18 cells**: 9 markdown cells, 9 code cells, and 5 saved outputs. The HTML view was generated from the existing notebook content so reviewers can inspect the work without relying on GitHub's notebook preview.

## Methods and tools shown

- Random Forest classification
- XGBoost classification
- TabPFN classifier usage
- One-hot encoding
- Train/test split
- Accuracy and classification-report evaluation
