## Overview

Introductory image classification notebook using PyTorch to train a multilayer perceptron on Fashion-MNIST clothing images. The notebook loads the dataset, visualises examples, defines an MLP, trains the model, and displays predicted versus actual outcomes.

## What the notebook contains

This notebook contains **11 cells**: 6 markdown cells, 5 code cells, and 11 saved outputs. The HTML view was generated from the existing notebook content so reviewers can inspect the work without relying on GitHub's notebook preview.

## Methods and tools shown

- PyTorch image classification
- Fashion-MNIST dataset loading
- MLP model design
- CrossEntropyLoss and Adam optimiser
- Prediction visualisation
- Basic model evaluation
