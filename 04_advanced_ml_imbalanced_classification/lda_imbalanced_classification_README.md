## Overview

Machine learning workflow for an imbalanced binary classification dataset using Linear Discriminant Analysis. The notebook loads and cleans structured data, drops high-missingness columns, imputes values, selects features, balances data with SMOTE and undersampling, and evaluates classification performance.
