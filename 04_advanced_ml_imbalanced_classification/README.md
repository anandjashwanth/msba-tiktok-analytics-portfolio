# Advanced ML for Imbalanced Classification

## Recommended application use
Use this for TikTok Shop EMEA Data Science and Data Analyst roles, but do not make it the first project for business strategy roles.

## Summary
This folder contains model benchmarking work for imbalanced binary classification using LightGBM, Random Forest, Balanced Random Forest and Linear Discriminant Analysis. It demonstrates practical model training, balancing strategies, feature selection and evaluation beyond accuracy.

## Skills shown
Python, LightGBM, Random Forest, Balanced Random Forest, LDA, feature selection, SMOTE, random undersampling, class imbalance, ROC AUC, precision, recall, F1-score, confusion matrix, threshold tuning.

## Files included
- `lightgbm_imbalanced_classification.ipynb` — LightGBM modelling and resampling comparison.
- `random_forest_imbalanced_classification.ipynb` — Random Forest / Balanced Random Forest workflow.
- `lda_imbalanced_classification.ipynb` — Linear Discriminant Analysis with feature selection and resampling.

## Note
The notebook originally labelled as logistic regression appears similar to the Random Forest workflow, so it has been placed in the supporting archive rather than this main ML folder.
