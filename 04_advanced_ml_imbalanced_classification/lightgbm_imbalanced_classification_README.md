# LightGBM Imbalanced Classification Benchmark

## Files

- Notebook: `lightgbm_imbalanced_classification.ipynb`
- HTML view: `lightgbm_imbalanced_classification.html`
- This README: `lightgbm_imbalanced_classification_README.md`

## Overview

LightGBM modelling notebook for an imbalanced binary classification problem. The notebook compares LightGBM trained with class weighting, SMOTE plus random undersampling, and random undersampling, then evaluates the effect on minority-class detection.

## What the notebook contains

This notebook contains **12 cells**: 2 markdown cells, 10 code cells, and 40 saved outputs. The HTML view was generated from the existing notebook content so reviewers can inspect the work without relying on GitHub's notebook preview.

## Methods and tools shown

- LightGBM classification
- Class weighting
- SMOTE and random undersampling
- Mutual-information-style feature selection
- ROC curve and AUC evaluation
- Precision, recall and F1 comparison
