# Statistical Business Analysis

## Recommended application use
Use this as supporting evidence for communication, statistical reasoning and stakeholder-ready analytical reporting.

## Summary
This folder contains statistical projects covering London Underground temperature analysis, household expenditure regression, podcast engagement analysis and NLP/topic modelling.

## Skills shown
R, tidyverse, regression, ANOVA, interaction modelling, hypothesis testing, data cleaning, descriptive statistics, visualisation, topic modelling, sentiment analysis, business interpretation.

## Files included
- `london_underground_temperature_analysis.Rmd` — London Underground temperature analysis.
- `business_statistics_regression_analysis.Rmd` — regression, ANOVA and interaction modelling.
- `fake_news_sentiment_topic_modeling.ipynb` — fake news sentiment and topic modelling.
