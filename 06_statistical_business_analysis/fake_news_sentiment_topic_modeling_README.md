## Overview

NLP analytics project analysing fake-news style text using transformer-based sentiment analysis and topic modelling. The notebook loads uploaded data, applies a Hugging Face sentiment pipeline, preprocesses text, and uses LDA to identify topic patterns.

## What the notebook contains

This notebook contains **22 cells**: 5 markdown cells, 17 code cells, and 24 saved outputs. The HTML view was generated from the existing notebook content so reviewers can inspect the work without relying on GitHub's notebook preview.

## Methods and tools shown

- Hugging Face transformer sentiment analysis
- Text preprocessing
- CountVectorizer
- Latent Dirichlet Allocation topic modelling
- Topic-word interpretation
- Visual and tabular NLP analysis
