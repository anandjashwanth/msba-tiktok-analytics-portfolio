# Which projects to submit for each TikTok internship

## Strategy & Analytics Project Intern — TikTok Shop
Submit these first:
1. Retail Inventory SQL Analytics for PopBeauty
2. Loan Policy A/B Testing and Borrower Segmentation
3. Revenue and Capacity Optimisation for Rental Operations
4. Business Statistics and Regression Analysis in R

Why: this role is about business reviews, target planning, deep-dives, competitor/market thinking, and recommendations. Lead with business diagnosis, SQL, experimentation and optimisation.

## Data Analyst Project Intern — TikTok LIVE
Submit these first:
1. Loan Policy A/B Testing and Borrower Segmentation
2. Deep Learning Default Prediction with Segment-Level Evaluation
3. Retail Inventory SQL Analytics for PopBeauty
4. Statistical Business Analysis and Regression Modelling

Why: this role asks for trend analysis, dashboards, OKRs, creator/user segmentation, predictive modelling and communication to non-technical stakeholders. The borrower segmentation project is the closest match to creator/user segmentation.

## Business Analytics Project Intern — TikTok Shop

## Data Analyst Project Intern — TikTok Shop EMEA Data Science
Submit these first:
1. Retail Inventory SQL Analytics for PopBeauty
