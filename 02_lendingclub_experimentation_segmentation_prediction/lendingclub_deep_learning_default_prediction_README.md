## Overview

PyTorch dense neural network project for predicting loan default risk using LendingClub-style credit variables. The notebook covers feature selection, leakage prevention, encoding, scaling, undersampling, train/validation/test splitting, model training, hyperparameter tuning, and evaluation.

## What the notebook contains

This notebook contains **44 cells**: 21 markdown cells, 23 code cells, and 21 saved outputs. The HTML view was generated from the existing notebook content so reviewers can inspect the work without relying on GitHub's notebook preview.

## Methods and tools shown

- PyTorch neural network modelling
- BatchNorm, ReLU and Dropout layers
- Class imbalance handling through undersampling
- Train/validation/test split
- ROC AUC, precision, recall and F1 evaluation
- Segment-level interpretation for credit-risk decisions
