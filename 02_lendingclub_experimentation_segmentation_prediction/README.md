# LendingClub Experimentation, Segmentation and Prediction

## Recommended application use
Use this as a headline project for TikTok LIVE Data Analyst and a strong supporting project for TikTok Shop Strategy & Analytics.

## Summary
This folder combines three connected analytics projects using LendingClub data: A/B testing of loan review policies, borrower segmentation with clustering, and deep learning default prediction. Together they show how to define metrics, compare policies, find customer segments, evaluate model performance and connect analysis to business decisions.

## Skills shown
A/B testing, KPI/OEC design, statistical testing, false positive/false negative analysis, clustering, segmentation, PyTorch, feature engineering, class imbalance handling, model evaluation, business recommendations.

## Files included
- `loan_policy_ab_testing_report.html` — A/B testing of loan review policies.
- `borrower_segmentation_cluster_analysis_report.html` — borrower segmentation via cluster analysis.
- `lendingclub_deep_learning_default_prediction.ipynb` — deep learning default prediction with segment-level evaluation.
