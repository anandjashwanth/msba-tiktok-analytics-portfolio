# MSBA TikTok Analytics Portfolio

This repository is a reviewed and GitHub-ready MSBA portfolio for TikTok Shop, TikTok LIVE, BytePlus AI Agent, and related analytics internship applications. It contains selected projects in SQL business analytics, experimentation, segmentation, optimisation, machine learning, statistical analysis, and AI-enabled data products.

Each Python notebook now has:

- the original `.ipynb` notebook,
- a generated `.html` view for easier reading when GitHub notebook preview is slow or blank,
- a notebook-specific `_README.md` explaining the objective, methods, relevance, and review notes.

## Viewing notebooks

GitHub sometimes fails to preview `.ipynb` files. Use the paired `.html` files or the detailed notebook README files when this happens. The full index is available in `HTML_NOTEBOOK_INDEX.md`.

## Rendered Portfolio Homepage

This repository includes an `index.html` page designed for GitHub Pages. After GitHub Pages is enabled, open the Pages homepage to view the rendered HTML notebook reports embedded directly on one page:

```text
https://YOUR-USERNAME.github.io/msba-tiktok-analytics-portfolio/
```

The normal GitHub file viewer will show `.html` source code. The GitHub Pages version renders the reports as webpages.
