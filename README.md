This portfolio is arranged for the following internship applications:

- Strategy & Analytics Project Intern — TikTok Shop

## Recommended submission approach

Do not attach every raw coursework file separately in the application form. Upload this folder structure to GitHub, GitHub Pages, Notion, or a clean portfolio site, then submit links to the strongest project folders.

## Before publishing

1. Remove API keys, tokens, credentials and private paths.
2. Remove or anonymise restricted datasets if permission is not available.
3. Label collaborative work honestly and describe your personal contribution.
4. Keep HTML/PDF reports for recruiters and notebooks for technical reviewers.
5. Add screenshots of key charts/tables to GitHub or Notion project pages.
